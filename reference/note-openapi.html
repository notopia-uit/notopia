List notes​

GET
/note/notes

Retrieve a paginated list of all notes with optional filtering and sorting

Authorizations​

oauth2

OAuth2 Security Scheme

authorizationCode Flow
Authorization URL"http://authentik.notopia.localhost/application/o/authorize"
Token URL"http://authentik.notopia.localhost/application/o/token"
Scopes:
  • "openid"Required for OIDC
  • "profile"Access to user profile
